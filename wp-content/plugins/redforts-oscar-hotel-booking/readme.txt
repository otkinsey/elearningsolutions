== Description ==

= Easy and fully integrated =
The Redforts Oscar Hotel Booking plugin allows you to easily integrate Redforts' Booking Engine on your Wordpress website and receive commission free bookings directly in the Oscar Property Management System (PMS). No more copying and pasting or setting availabilities and prices. Oscar does that all for you.

Redforts Software offers an all-in-one hotel software solution: a Property Management System (PMS) with fully integrated Internet Booking Engine and OTA Channel Manager connected to over 65 channels.

= Features =
* Fully integrated in Oscar Hotel Software
* Quick and easy setup
* Fully customisable colours
* Multiple language support

== Installation ==

= Installation through the Wordpress control panel =

1. Login to your Wordpress account using the administrator's user name and password.
2. Click the Plugins icon on the left menu.
3. Click the "Add New" button at the top of the Plugins menu.
4. Search for "Redforts Oscar Hotel Booking".
5. Click on "Install".
6. When the "Plugin installed successfully" message appears, click the "Activate Plugin" link.
7. Click on the main menu option with the Redforts logo.
8. Insert your client code and choose your colours.
9. Insert the tag [rdf-booking-widget] on the page you would like the widget to be displayed.

= Installation via FTP =

1. Download the plugin.
2. Upload the unzipped plugin folder to your Wordpress plugin folder (wp-content/plugins).
3. Go to Wordpress and click the Plugins icon on the left menu.
4. Activate the plugin.
5. Click on the main menu option with the Redforts logo.
6. Insert your client code and choose your colours.
7. Insert the tag [rdf-booking-widget] on the page you would like the widget to be displayed.

Yes, but you need a Oscar Hotel Software account.
